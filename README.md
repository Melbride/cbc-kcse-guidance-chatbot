# CBC-KCSE Guidance Chatbot

A comprehensive guidance platform for both CBC (Competency-Based Curriculum) and KCSE (Kenya Certificate of Secondary Education) students.

## Structure

- **kcse/**: KCSE Guidance Chatbot frontend
- **cbc-guidance-chatbot/**: CBC Guidance Chatbot frontend

## Deployment

Deployed on Vercel: https://shuleguideai.vercel.app/

### Admin Access
- KCSE Admin: https://shuleguideai.vercel.app/kcse/frontend/admin.html
- CBC Admin: https://shuleguideai.vercel.app/cbc-guidance-chatbot/frontend/admin.html
